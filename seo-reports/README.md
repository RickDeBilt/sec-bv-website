# seo-reports/

Analyse-rapporten op basis van de exports in [`../seo-data/`](../seo-data/).

## Wat hier hoort

Per analyseronde één markdown-rapport met:

- queries met veel vertoningen maar lage CTR;
- pagina's op positie 8-30 (de dichtstbijzijnde winst);
- ontbrekende zoekintenties;
- concrete voorstellen voor betere titles, meta descriptions en interne links.

## Naamgeving

```
rapport-2026-07-17.md
```

## Belangrijk

Een rapport bevat **voorstellen**, geen doorgevoerde wijzigingen. Niets uit een rapport
gaat live zonder akkoord. Zie de workflow in [`../seo-monitoring.md`](../seo-monitoring.md).
